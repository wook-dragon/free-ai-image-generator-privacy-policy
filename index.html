개인정보 처리방침

최종 업데이트: 2025년 6월 11일

1. 수집하는 개인정보

Free AI Image Generator 앱은 다음과 같은 개인정보를 수집할 수 있습니다:

  • 기기 정보 (기기 모델, OS 버전)
  • 이미지 생성 프롬프트 (사용자가 입력한 텍스트)

※ 다음 정보는 수집하지 않습니다:

  • 사용자 계정 정보
  • 생성된 이미지 (서버에 저장되지 않음)
  • 사용자 사용 통계
  • 광고 ID
  • 기기 ID

2. AI 이미지 생성 서비스 관련 개인정보 처리

본 앱은 FLUX.1-schnell AI 모델을 활용한 이미지 생성 서비스를 제공합니다:

  • 이미지 생성 시 입력하신 텍스트 프롬프트는 FLUX.1-schnell API 서버로 전송되어 이미지 생성에 사용됩니다.
  • 생성된 이미지는 서버에 저장되지 않으며, 사용자의 기기에만 저장됩니다.
  • 이미지 생성은 FLUX.1-schnell의 정책과 필터링 규칙을 따릅니다.
  • 생성된 이미지의 저작권 및 사용 권한은 FLUX.1-schnell의 정책을 따릅니다.

3. 광고 서비스 관련 개인정보 처리

본 앱은 Google AdMob을 통해 광고 서비스를 제공합니다:

  • 이미지 생성 및 다운로드 시 광고가 표시됩니다.
  • AdMob은 광고 제공을 위해 다음과 같은 정보를 수집할 수 있습니다:
    • 광고 식별자
    • 기기 정보
    • IP 주소
  • AdMob의 개인정보 처리방침은 Google 개인정보 처리방침을 따릅니다.

4. 개인정보의 수집 및 이용목적

수집된 개인정보는 다음의 목적을 위해 이용됩니다:

  • AI 이미지 생성 서비스 제공
  • 광고 서비스 제공
  • 앱 서비스 개선 및 최적화
  • 기술적 문제 해결

5. 개인정보의 보유 및 이용기간

개인정보는 원칙적으로 개인정보의 수집 및 이용목적이 달성되면 지체 없이 파기합니다:

  • 이미지 생성 프롬프트: 이미지 생성 완료 후 즉시 삭제
  • 생성된 이미지: 사용자 기기에만 저장되며, 사용자가 직접 삭제할 때까지 보관
  • 광고 관련 데이터: AdMob의 데이터 보관 정책에 따름

6. 개인정보의 제3자 제공

당사는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

  • FLUX.1-schnell API 서비스 제공을 위해 필요한 최소한의 정보만 전송
  • Google AdMob을 통한 광고 서비스 제공
  • 법령에 따른 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

7. 이용자의 권리와 행사방법

사용자는 언제든지 등록된 개인정보에 대해 다음의 권리를 행사할 수 있습니다:

  • 개인정보 열람 요구
  • 오류 정정 요구
  • 삭제 요구
  • 처리정지 요구
  • 생성된 이미지 삭제 요구

10. AI 서비스 이용 시 주의사항

AI 이미지 생성 서비스 이용 시 다음 사항을 주의해 주시기 바랍니다:

  • 부적절하거나 해로운 콘텐츠 생성은 FLUX.1-schnell의 필터링 정책에 따라 제한될 수 있습니다.
